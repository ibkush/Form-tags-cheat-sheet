HTML Forms Tags

Form Tags

Tags Name Description
<form> Form It is used to collect and send information to an external source.
<select> Select It can be used to create a dropdown list.
<input> Input It is used to render a variety of input fields on a webpage(textfields,checkboxes,buttons etc)
<textarea> Textarea It is used when creating a text-box for multi-line input (e.g a comment section)
<datalist> Datalist It is used when providing users with a list of pre-defined options, but also allows them to write alternative inputs as well(used with the input element).
<label> Label It is used to provide identification for aspecific input based on matching values for the input 'id' and label's 'for' attributes.

Form Attributes

Attibute Name Description
name Name It is the key-value pair that enables the form to send data.
required Required It is used to specify that a field must include a value.
max Maximum It specifies the maximum value for the input class.
maxlength Maximum length It spcifies the maximum number of characters that can be entered into a textfield.
pattern Pattern It uses a regular expression to match against(or validate) the value of an input when a form is submitted.
minlength Minimum length It specifies the minimum number of characters that can be entered into a textfield.
min Minimum it specifies the minimum value for the input class.
